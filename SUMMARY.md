# 项目总结（MD）

## 一、研究目标
本项目围绕“二三维城市结构对季节性地表短波净辐射（NSSR）的影响”展开，重点解决：
1. 四季 NSSR 的稳定反演
2. 基于地物类型的二维定量评估
3. 基于城市形态因子的季节解耦机器学习建模与 SHAP 解释

## 二、已完成工作
### 1) NSSR 反演链路
- 已统一采用短波净辐射公式：`NSSR = DSSR * (1 - Albedo)`
- ERA5 数据源已切换为 `era52` 优化版
- 输出四季反演结果（Spring/Summer/Autumn/Winter）

### 2) 二维地物定量评估
- 使用 NSSR 与 ESA LC30 数据完成分区统计
- 输出：季节均值、标准差、像元占比、贡献率、核心区/郊区差异
- 输出图件：季节均值图、范围图、标准化对比图、空间分布图

### 3) 三维因子建模与解释
- 完成最新 `output2` 训练样本生成与四季解耦训练
- 使用 SHAP 分析主导因子，结果显示 SVF 持续主导，FAR/BSA/MBH 次之

## 三、阶段性结论
1. 新版 NSSR 季节律合理，且与验证产品的 DSSR 对比显著改善
2. 二维分析可作为三维建模的空间参照基线
3. 当前 R² 相比旧版本下降，主要来自目标定义更严格（NSSR）与空间泛化难度提高，不是“模型污染”
